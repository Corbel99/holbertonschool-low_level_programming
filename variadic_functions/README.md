# Variadic Functions

## Description

This project is part of the Holberton School Low Level Programming curriculum.

The goal of this project is to understand and use **variadic functions** in C. It introduces the `<stdarg.h>` library and the macros used to access a variable number of arguments passed to a function.

## Learning Objectives

At the end of this project, I am able to explain:

- What variadic functions are.
- How to use `va_list`.
- How to use `va_start`.
- How to use `va_arg`.
- How to use `va_end`.
- Why and how to use the `const` qualifier.

## Requirements

- Ubuntu 20.04 LTS
- GCC
- Compilation flags:
  ```
  gcc -Wall -Werror -Wextra -pedantic -std=gnu89
  ```
