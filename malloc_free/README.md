# C - malloc, free

## Description

This project introduces dynamic memory allocation in C.

The goal is to learn how to allocate memory during program execution using `malloc()`, how to release it with `free()`, and how to avoid memory leaks.

During this project, I will learn the difference between automatic and dynamic memory allocation, when to use each one, and how to manage memory correctly.

---

## Learning Objectives

At the end of this project, I should be able to explain:

- The difference between automatic and dynamic allocation.
- How `malloc()` works.
- How and when to use `free()`.
- Why memory leaks are a problem.
- How to use Valgrind to detect memory leaks.
