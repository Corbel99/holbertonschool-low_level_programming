# More malloc, free

## Description

This project is part of the Holberton School Low-Level Programming curriculum.

The objective is to deepen the understanding of dynamic memory allocation in C by using `malloc` and `free`, while recreating the behavior of some standard library functions.

## Learning Objectives

- Dynamic memory allocation
- Memory management
- `malloc`
- `free`
- `exit`
- Recreating `calloc`
- Recreating `realloc`

## Requirements

- Ubuntu 20.04 LTS
- gcc
- Compilation flags:

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89
```

## Files

| File | Description |
|------|-------------|
| `0-malloc_checked.c` | Allocates memory and exits on failure |
| `1-string_nconcat.c` | Concatenates two strings using n bytes |
| `2-calloc.c` | Recreates the behavior of `calloc` |
| `3-array_range.c` | Creates an array of integers |
| `100-realloc.c` | Recreates the behavior of `realloc` |
