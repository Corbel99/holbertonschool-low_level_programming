# Variables, if, else, while

This project introduces the basics of the C programming language, including variables, data types, conditional statements (`if`, `else`, `else if`), and loops (`while`).

## Learning Objectives

* Understand variables and data types in C
* Use arithmetic and comparison operators
* Write conditional statements with `if`, `else`, and `else if`
* Create loops using `while`
* Compile C programs with GCC
* Follow the Betty coding style

## Requirements

* Ubuntu 24.04 LTS
* GCC compiler
* Betty style checker
* Programs compiled with:

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89
```
