# Pointers, Arrays and Strings

## Description

This project introduces the fundamentals of pointers, arrays, and strings in C. It focuses on understanding how memory addresses work, how to manipulate data using pointers, and how arrays and strings are stored and accessed in memory.

The project also covers the scope of variables and the relationship between pointers and arrays through practical exercises.

## Learning Objectives

By the end of this project, I should be able to explain:

- What pointers are and how to use them.
- What arrays are and how to use them.
- The differences between pointers and arrays.
- How strings are represented and manipulated in C.
- The scope of variables.

## Requirements

- Ubuntu 20.04 LTS
- GCC with the following flags:
  - `-Wall`
  - `-Werror`
  - `-Wextra`
  - `-pedantic`
  - `-std=gnu89`
- Betty coding style
- No global variables
- Maximum of 5 functions per file
- Only `_putchar` is allowed for output

## Files

| File | Description |
|------|-------------|
| `0-reset_to_98.c` | Updates the value pointed to by a pointer to `98`. |
